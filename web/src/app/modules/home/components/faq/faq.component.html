Frequently Asked Questions:

Is PDF Box Free?

Yes! PDF Box is completely free. Revenue is generated from ads, so please don't block them.

Do I need to make an account?

No. However, if you'd like to save all of the documents you convert you need an account.

How do I use PDF Box?

Simply upload a document and click convert!

What types of files can PDF Box convert?

PDF Box currently converts Microsoft Word, Excel, and Powerpoint.
